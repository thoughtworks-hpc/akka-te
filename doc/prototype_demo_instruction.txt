1. Start a cluster of 2 nodes:
mvn exec:java -Dexec.mainClass="com.thoughtworks.hpc.te.demo.prototype.MatchEngineServer" -Dexec.args="50051 25251 SymbolA"
mvn exec:java -Dexec.mainClass="com.thoughtworks.hpc.te.demo.prototype.MatchEngineServer" -Dexec.args="50052 25252 SymbolB"

2. Connect to nodes via grpc port 50051 or 50052

3. Send rpc request with value like:
    symbol: "SymbolA"
    price: 7
    quantity: 27
    side: 1
